A. Soal Subneting IP Address

Soal dan jawaban jaringan komputer pertemuan 3 mengenai subneting IP Address kelas A, B, dan C

  1. IP Kelas A:

    1. 44.10.0.1/8
      • Biner : 11111111.00000000.00000000.00000000
      • Network : 20 = 1 Network
      • Host : (224) - 2 = 16777214 Host
    2. 10.11.12.2/10
      • Biner : 11111111.11000000.00000000.00000000
      • Network : 22 = 4 Network
      • Host : (222) - 2 = 4194302 Host

  2. IP Kelas B:

    1. 175.23.15.0/21
      • Biner : 11111111.11111111.11110000.00000000
      • Network : 24 = 16 Network
      • Host : (212) - 2 = 4094 Host
    2. 175.23.75.0/17
      • Biner : 11111111.11111111.10000000.00000000
      • Network : 21 = 2 Network
      • Host : (215) - 2 = 32766 Host

  3. IP Kelas C:

    1. 192.168.12.6/26
      • Biner : 11111111.11111111.11111111.11000000
      • Network : 22 = 4 Network
      • Host : (26) - 2 = 62 Host
    2. 192.168.18.3/29
      • Biner : 11111111.11111111.11111111.11110000
      • Network : 24 = 16 Network
      • Host : (24) - 2 = 14 Host

B. Perangkat Jaringan Komputer

Di era modern saat ini, kebutuhan internet sudah banyak dan menjadi kebutuhan sehari-hari untuk menyelesaikan pekerjaan manusia. Oleh karena itu, dibutuhkan perangkat-perangkat jaringan yang mendukung aktivitas komunikasi di internet. Perangkat-perangkat jaringan yang akan saya bahas pada pertemuan saat ini, diantaranya adalah Repeater, Bridge, dan Network Interface Card (NIC).

1. Repeater

Image - Repeater

Repeater adalah salah satu perangkat jaringan yang berfungsi untuk memperluas jaringan atau sinyal Wi-Fi. Terdapat 2 tipe repater, yaitu repeater untuk memperkuat sinyal wireles atau Wi-Fi, dan juga Repeater untuk jaringan kabel.

Fungsi repeater kabel, juga memiliki fungsi yang sama, yaitu memperluas jaringan atau memperkuat sinyal yang di transmisikan, dari repeater Wi-Fi dengan menambahkan repeater jenis kabel agar data yang di transmisikan tetap stabil dengan jangkauan yang jauh.

Fungsi Repeater

Repeater memiliki beberapa fungsi atau kegunaan yang bisa dimanfaatkan dalam jaringan komputer, di antaranya:

  1. Memperluas jangkauan area sinyal.
  2. Kemudahan akses yang baik dari server.
  3. Memperkuat sinyal yang lemah dari jaringan server.

Jenis-Jenis Repeater

Terdapat beberapa jenis Repeater, diantaranya:

  1. Repeater Telepon (Telephone Repeater)
    Jenis repeater ini akan memperkuat sinyal telepon yang lemah, karena faktor jarak yang mungkin sangat jauh.

  2. Radio Repeater
    Jenis Repeater ini, memiliki beberapa antena yang berguna sebagai receiver dan transmitter. Selain itu, Radio Repeater berfungsi untuk memperkuat sinyal radio ataupun jaringan Wi-Fi.

Repeater menggunakan 2 sistem dalam menjalankan fungsinya, yaitu sistem analog dan digital. Sistem analog dalam repeater dapat melakukan transmisi data berupa data analog. Sedangkan Repeater dengan sistem digital, akan mengirimkan data digital yang dikonversi dalam angka binary.

2. Bridge

Image - Bridge Network Device (Source EasyTechJunkie.com)

Bridge adalah perangkat jaringan yang berfungsi untuk menghubungkan beberapa jaringan LAN yang sama atau berbeda. Bridge berada pada data-link layer pada OSI (Open System Interconnetion). Itulah yang membuat bridge bekerja dengan menyambungkan jaringan komputer dengan access control yang berbeda serta mempelajari alamat link dari perangkat yang terkoneksi pada jaringan yang sama.

Image - Bridge Network Illustration

Fungsi Bridge

  1. Menghubungka beberapa jaringan LAN yang sejenis, sehingga memiliki kapasitas yang besar.
  2. Menghubungkan beberapa jaringan yang berada di area terpisah, meski dengan tipe jaringan yang berbeda.
  3. Bisa berfungsi sebagai router.

Bridge memiliki beragam fungsi, seperti diantaranya, memetakan alamat/link dari ethernet, ke beberapa node masing-masing device sesuai lalu lintas bridge yang diperlukan.

3. Network Interface Card (NIC)

Image - Network Interface Card (NIC)

Network Interface Card atau NIC adalah perangkat jaringan yang berbentuk seperti papan elektronik (PCB) yang berfungsi untuk mengubah atau mengkonversi sinyal jaringan komputer menjadi data yang dapat dikirim atau diterima dalam suatu jaringan.

NIC atau LAN Card sesuai namanya berbentuk seperti kartu yang memiliki slot, dan di pasang didalam komponen slot motherboard komputer. NIC memiliki alamat khusus yang disebut dengan MAC address, yang berupa kumpulan kode 48bit dalam bentuk heksa, yang hanya dimiliki oleh satu device NIC, sehingga NIC memiliki MAC Address yang berbeda-beda. Hal ini karena, NIC memungkinkan di pakai atau digunakan pada kartu ethernet lainnya, yang akan menyebabkan collision atau bentrokan data, sehingga NIC wajib memiliki suatu kode unik dari setiap devicenya.

Fungsi NIC

  1. Sebagai alat jaringan yang berfungsi mengirimkan dan menerima data melalui jaringan kabel.
  2. Kontrol aliran transmisi data pada sistem kabel jaringan.
  3. Sebagai penerima data dari komputer lain melalui kabel, dengan mengkonversi/memecah data ke dalam bit.

Cara Kerja NIC

Beberapa cara kerja NIC, dalam jaringan komputer:

  1. NIC akan memecah data menjadi Bit, agar komputer mudah mengenalinya.
  2. Setelah memecah data menjadi bit, NIC akan kembali merangkainya dari bit menjadi data yang utuh.
  3. NIC akan mengenali kode MAC address (Media Access Control), pada setiap NIC agar diharapkan untuk menghindari konflik data dalam sistem jaringan komputer.
  4. Node jaringan akan mengirimkan paket data tertentu, ini tergantung apakah jaringan tersebut sedang mengirim paket data atau tidak. Jika tidak, maka node ini yang akan mengirimkan paket datanya.
  5. Node akan berhenti bersamaan dan tidak mengirimkan paket data.
  6. Node akan menunggu waktu agar dapat mengirimkan paket data lagi.
  7. Paket data yang terjadi collision akan dikirimkan kembali pada saat ada kesempatan.

Jenis NIC

  1. NIC Fisik
    NIC jenis ini berbentuk seperti card/kartu yang memiliki slot yang di pasangkan pada motherboard komputer. Pada NIC ini juga terdapat port RJ-45 untuk mengkoneksikan ke jaringan dengan kabel.

  2. NIC Logis
    NIC ini tidak memiliki bentuk fisik, melainkan software yang mendefinisikan bahwa itu merupakan sebuah NIC jaringan.
